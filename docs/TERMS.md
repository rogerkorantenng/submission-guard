## Description of Service

Submission Guard is a Reddit moderation tool that automatically enforces submission rules including:
- Title tag validation
- NSFW-in-title detection
- Paragraph length limits
- Code block detection
- Rate limiting
- Series post auto-flair

The App operates on subreddits where it is installed by moderators.
